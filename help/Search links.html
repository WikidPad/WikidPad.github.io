Description

General

Only works in Edit mode
Does create a link in View mode, but acts as a standard link to a page.

Searches for the first occurrence of the text in an item and goes there
Searches for text as it appears in the Edit window
Searchtext can include textformatting and other characters
Is case insensitive
Reserved chars and spaces have to be preceded with a "#"

Format

WikiWord#search-text
[link]#search-text

Examples

OptionsDialog#*Sort# order*
[Options HTML preview_export]#*Preview# renderer*
See OptionsDialog#TimeView# Pos
