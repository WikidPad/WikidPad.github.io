General

You can add links to websites to your page and open them in your browser. You do this with a default web URL.

  • http://groups.google.com/group/wikidpad/web/home
  • http://tech.groups.yahoo.com/group/wikidPad/
  • http://wikidpad.python-hosting.com/
  • http://en.wikipedia.org/wiki/WikidPad

Link titles

Text

A link text can in preview and export be replaced by a link title, that then serves as a clickable synonym for the link.
  • Wikidpad: Home page
  • Wikidpad: Yahoo user group
  • Wikidpad: Trac home page
  • Wikidpad: On Wikipedia

Images

Instead of a text, also an image can be used as a "title" or synonym for a link.
  • Wikidpad: Home page
  • Wikidpad: Yahoo user group
  • Wikidpad: Trac home page
  • Wikidpad: On Wikipedia
The images can be formatted with the default image attributes; see Textformatting: images.
