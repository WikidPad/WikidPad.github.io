Options: Menu settings

This page lists all the possible settings in the main menu [1].

Top
[1] The page has been assembled using page inserts; it's contents can only be seen in preview.

Wiki words

    Wiki words: clipboard catcher
• Clipboard catcher at page
Switches the clipboard catcher on and appends copied text to the page for which it was activated.

• Clipboard catcher at cursor
Switches the clipboard catcher on and inserts copied text at the current cursor position.

• Clipboard catcher off
Switches the clipboard catcher off.

Top

Editor

    Editor
• Set date format
Set the format for the "insert date" menu option. Shows a dialog window with a directive list and a preview section with whitch the date format can be set.

Top
    Editor: settings, wrap
• Wrap mode
Switches wrap mode on or off.
    Editor: settings, indent
• Auto-indent
Switches auto-indentation on or off
    Editor: settings, bullets
• Auto-bullets
Switches the automatic creation of new list items in the editor on or of.
    Editor: settings, tab key
• Tabs to spaces
Controls whether tabs are inserted as tabs or as spaces.

Top

Folding

    Folding
• Show folding
Switches the support for folding in the editor on or off.

Top

View

    View: workspace
• Show toolbar: Ctrl-Shift-T
Switches the display of the toolbar on or of.

• Stay on top
Set Wikidpad to be always on top of the other windows, or undo that setting.
    View: main tree
• Show tree control: Ctrl-T
Show or hide the main tree window. Very usefull when the tree option "Tree auto-hide" has been set to true.
    View: structure view
• Show time view
Show or hide the document structure view window.
    View: time view
• Show time view
Show or hide the time view window.
    View: editor
• Show indentation guides
Switches the display of indentation guides in the editor on or of.

• Show line numbers
Switches the display of line numbers in the editor on or of.
