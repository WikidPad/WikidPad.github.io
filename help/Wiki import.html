General

Currently import only supports importing MultipageTextFormat. To call up the import dialog, select the option "Wiki / Import.." from the main menu.

The dialog window

• Import format
Here you choose the type of import you want. Currently only Multipage text is supported.

• Source file/directory
Here you select the file to import.
