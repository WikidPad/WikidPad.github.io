General

The Log window is Wikidpad's message area. Wikidpad continuously checks pages for changes and possible errors. Specifically when there are errors in the area of attributes, the log window will appear and a message will be displayed. Like in the screenshot below, where the icon attribute has been specified twice.

The log window normally is hidden - default setting - it will only appear, when there are messages to show.

Double clicking a message brings the editor to the offending attribute. Position may be a bit off if page was modified after saving.

Currently only the attributes alias, color, icon and bold are checked for errors. The presentation attributes inside todo items are not checked.

• Auto-show log window
Automatically show log window if new error messages arrive. If the check-box is gray, the application level setting "Auto-show log window" is used, otherwise this wiki level setting overrules it.
