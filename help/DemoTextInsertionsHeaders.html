Remarks

  • The [:toc:] in the inserting page is not updated for headers in the inserted text; nór in preview nór in html export
  • The [:rel:] link insertions áre updated for the inserting page
  • You cannot create "Top" links using regular anchor references, since these need the name of the page they are on; the inserting page in this case[1].

Top

Footnotes:
[1] You can create "Top" links for "insert pages", by using html tags with an empty anchor name; like this: <a href="#">Top</a>. Anchors with empty names default to "Top of page". Attention: this does not work in the internal preview renderer.
